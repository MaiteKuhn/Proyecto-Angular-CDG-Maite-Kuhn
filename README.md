Temática del proyecto:
El proyecto consiste en el desarrollo de una aplicación web para el Club Deportivo Garmense, una institución deportiva.
El objetivo de la aplicación es brindar un espacio digital donde socios, deportistas y simpatizantes puedan conocer la historia del club, sus diferentes disciplinas deportivas y las actividades que se desarrollan dentro de la institución.
Además, contará con una sección de venta de indumentaria oficial del club, donde los usuarios podrán visualizar los productos disponibles y consultar información relevante para la compra, como talle, precio y disponibilidad de stock.
También se incorporará una sección destinada a aquellas personas que deseen asociarse a la institución.
